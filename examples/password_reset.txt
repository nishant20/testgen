As a registered user, I want to reset my password via a link sent to my email
so that I can regain access to my account if I forget my password.

Acceptance criteria:
- A "Forgot password" link is available on the login page.
- Submitting a registered email sends a reset link that is valid for 30 minutes.
- The reset link opens a form to set a new password.
- The new password must meet the password policy: minimum 8 characters, at least
  1 number and 1 uppercase letter.
- After a successful reset, the user can log in with the new password and the old
  password no longer works.
- Submitting an unregistered email shows the same generic confirmation message
  (no account enumeration).
- A reset link can only be used once.
